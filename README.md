# Grok 4 Arena — Wave Defense

**Playable 3D first-person wave shooter** inspired by (and aiming to match or exceed) the Grok 4 Gaming demo shown at the xAI Grok 4 unveil.

## How to Play

1. Open `index.html` in Chrome or Edge (desktop recommended).
2. Click anywhere in the game to lock the pointer (FPS controls).
3. **Controls**:
   - **WASD** — Move
   - **Mouse** — Look around
   - **Left Click** (hold) — Shoot (auto-fire while held)
   - **Hold Shift** — Boost (speed boost + particle trail, drains energy)
   - **ESC** — Unlock mouse

4. Survive waves of astral drones. Kill all enemies in a wave to advance. Use crates and barrels for cover.

## Features (Grok 4 Quality)

- High-fidelity space nebula arena with procedural textures
- Visible first-person weapon with recoil
- Dynamic wave progression + increasing difficulty
- Destructible cover (shoot the crates!)
- Full particle systems, muzzle flashes, impacts, boost trails
- Web Audio generated laser / explosion / boost sounds
- Smooth controls and polished HUD
- "Grok-sourced" aesthetic: All textures and models are procedurally generated in the style of Three.js examples + web 3D assets

## Technical Notes

- 100% self-contained single HTML file (Three.js r134 via CDN)
- No build step, no dependencies to install
- Runs locally at 60fps on modern hardware
